 (require OptionsTradingDSL) package: OptionsTradingDSL

This library provides functions for creating, analyzing, and visualizing options trading strategies. It allows users to model complex option strategies and visualize their profit/loss profiles over time and price ranges.

procedure

(make-strategy/shortcut name    
  #:ticker ticker    
  #:ticker-price cp    
  [#:quantity qty    
  #:volatility vol    
  #:risk-free-rate rfr]    
  strategy-type    
  args ...) → strategy?
  name : any/c
  ticker : symbol?
  cp : positive?
  qty : positive-integer? = 1
  vol : (>=/c 0) = 0.2
  rfr : (>=/c 0) = 0.05
  strategy-type : any/c
  args : any/c
Creates an option strategy using a shortcut method for predefined strategy types.

The name parameter provides a descriptive name for the strategy.

The ticker parameter specifies the underlying security symbol.

The cp parameter (ticker-price) specifies the current price of the underlying security.

The qty parameter specifies the quantity of the strategy to create (default: 1).

The vol parameter specifies the implied volatility as a decimal (default: 0.2).

The rfr parameter specifies the risk-free interest rate as a decimal (default: 0.05).

The strategy-type and additional args define the specific option strategy to create (e.g., spread types, strike prices, expirations).

Returns a strategy object representing the complete options strategy.

(make-strategy name    
  #:ticker ticker    
  #:ticker-price cp    
  [#:volatility vol    
  #:risk-free-rate rfr]    
  legs ...) → strategy?
  name : any/c
  ticker : symbol?
  cp : positive?
  vol : (>=/c 0) = 0.2
  rfr : (>=/c 0) = 0.05
  legs : any/c
Creates an option strategy by explicitly specifying each option leg.

The name parameter provides a descriptive name for the strategy.

The ticker parameter specifies the underlying security symbol.

The cp parameter (ticker-price) specifies the current price of the underlying security.

The vol parameter specifies the implied volatility as a decimal (default: 0.2).

The rfr parameter specifies the risk-free interest rate as a decimal (default: 0.05).

The legs parameter is a list of option legs, which can be created using helper functions like buy and sell (not shown in the provided code).

Returns a strategy object representing the complete options strategy.

(graph-multiple-strategies-3d strategies    
  [#:min-price min-price    
  #:max-price max-price    
  #:min-days min-days    
  #:max-days max-days    
  #:price-step price-step    
  #:day-step day-step]) → void?
  strategies : (listof (list/c strategy? any/c any/c))
  min-price : (or/c #f real?) = #f
  max-price : (or/c #f real?) = #f
  min-days : real? = 1
  max-days : real? = 60
  price-step : real? = 5
  day-step : real? = 2
Creates a 3D visualization of multiple option strategies showing profit/loss over both stock price and time dimensions.

The strategies parameter is a list of triplets, where each triplet contains: a strategy object, a label for the legend, and a color for the plot.

The min-price and max-price parameters define the price range for the visualization. If not provided, appropriate bounds will be calculated.

The min-days and max-days parameters define the time range (in days) for the visualization.

The price-step and day-step parameters control the granularity of the visualization.

Opens a new window with the 3D visualization.

Creates a 2D visualization of multiple option strategies showing profit/loss profiles at a specific point in time.

The strategies parameter is a list of triplets, where each triplet contains: a strategy object, a label for the legend, and a color for the plot.

The min-price and max-price parameters define the price range for the visualization. If not provided, appropriate bounds will be calculated.

The days-since-purchase parameter specifies the time point (in days since purchase) for which to display the profit/loss profile. If not provided, a default value will be used.

procedure

(graph-decision strategy-triplets    
  [#:3d use-3d?    
  #:min-price min-price    
  #:max-price max-price    
  #:min-days min-days    
  #:max-days max-days    
  #:price-step price-step    
  #:day-step day-step    
  #:days-since-purchase days-since]) → void?
  strategy-triplets : (listof (list/c strategy? any/c any/c))
  use-3d? : boolean? = #f
  min-price : (or/c #f real?) = #f
  max-price : (or/c #f real?) = #f
  min-days : real? = 1
  max-days : (or/c #f real?) = #f
  price-step : real? = 5
  day-step : real? = 2
  days-since : (or/c #f real?) = #f
A convenience function that chooses between 2D and 3D visualizations based on the use-3d? parameter.

The strategy-triplets parameter is a list of triplets, where each triplet contains: a strategy object, a label for the legend, and a color for the plot.

The use-3d? parameter determines whether to create a 3D visualization (when #t) or a 2D visualization (when #f).

The remaining parameters are passed to either graph-multiple-strategies-3d or graph-multiple-strategies-2d based on the visualization type.

If max-days is not provided, it will be automatically calculated based on the maximum expiration date across all strategies.

Opens a new window with the chosen visualization.

3 ExamplesðŸ”—â„¹

Here are some examples of using this library to create and visualize option strategies:

#lang racket
(require OptionsTradingDSL)
 
;; Create a long call strategy
(define long-call
  (make-strategy/shortcut 'long-call
                        #:ticker 'AAPL
                        #:ticker-price 150
                        #:quantity 1
                        #:volatility 0.25
                        'long-call 160 30))
 
;; Create a bull call spread
(define bull-spread
  (make-strategy/shortcut 'bull-spread
                        #:ticker 'AAPL
                        #:ticker-price 150
                        #:quantity 1
                        #:volatility 0.25
                        'bull-call-spread 140 160 45))
 
;; Visualize these strategies
(graph-decision
  (list
    (list long-call "Long Call" 'blue)
    (list bull-spread "Bull Call Spread" 'red))
  #:3d #t
  #:min-price 130
  #:max-price 180)

Note: The exact available strategy types and leg creation helpers (like ’long-call, ’bull-call-spread) are not fully documented in the provided code. Consult the implementation for all available options or refer to additional documentation.
